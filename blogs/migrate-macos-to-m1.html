macOS迁移到M1处理器

Author:Gao
Created At:2022-03-10

将主力机更新到了M1处理器的macbook pro, 记录流程.

1. 操作系统更新到想同版本.

2. 运行migration assisstant, 将数据进行迁移

3. 迁移完成后需要处理的工作

由于macOS更换了处理器架构, 所以迁移的软件都是x86软件, 很多需要重新安装

homebrew的软件都需要重新安装

  1. 清除homebrew的formula和cask
for n in $(brew list); do echo $n; brew uninstall --ignore-dependencies --force $n; done

for c in $(brew list --cask)
do
  brew uninstall --cask --force $c;
done
  1. 删除目录
rm -rf /usr/local/Cellar
rm -rf /usr/loca/Caskroom
  1. 清空环境变量并删除homebrew
sudo rm -rf /usr/local/Homebrew
  1. 重新安装homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

有别于x86, arm版本homebrew会安装在目录/opt/homebrew

之后可以在安装之前需要安装的brew工具了

docker 报错 devmapper : Thin Pool

Author:Gao
Created At:2021-12-08

今天使用docker build镜像的时候出现错误：

devmapper: Thin Pool has 162422 free data blocks which is less than minimum required 163840 free data blocks. 
Create more free space in thin pool or use dm.min_free_space option to change behavior

解决方式：


docker rm $(docker ps -q -f status=exited)
docker volume rm $(docker volume ls -qf dangling=true)
docker rmi $(docker images --filter "dangling=true" -q --no-trunc)
